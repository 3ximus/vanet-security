------------------------

## Usage

Install and Run Certificate Authority

```
./mvn_script clean install ca
./mvn_script ca
```

Install and run the Roadside Unit

```
./mvn_script clean install rsu
./mvn_script rsu
```

Run vehicles (Simulates a new connected vehicle in the vanet)

```
./mvn_script vehicle
```
